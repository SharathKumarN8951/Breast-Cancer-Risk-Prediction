# Breast Cancer Risk Prediction

This project is a machine learning-based web application developed to predict the risk of breast cancer using patient health data. The goal of this project is to demonstrate how machine learning models can be applied in healthcare to assist with early risk assessment.

The application is built using Python and Streamlit, allowing users to enter the required information through a simple interface and receive a prediction instantly.

## Features

* Predict breast cancer risk based on input data
* Simple and interactive user interface
* Pre-trained machine learning model
* Real-time predictions
* Easy to run locally

## Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy
* Pickle

## Project Files

* `app.py` – Main Streamlit application
* `Breast Cancer Risk Prediction.ipynb` – Model development and experimentation
* `mortality_model.pkl` – Trained machine learning model
* `scaler.pkl` – Data scaling object
* `features.pkl` – Feature information used by the model
* `requirements.txt` – Required Python packages

## Getting Started

Clone the repository:

```bash
git clone https://github.com/SharathKumarN8951/Breast-Cancer-Risk-Prediction.git
```

Move to the project folder:

```bash
cd Breast-Cancer-Risk-Prediction
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

## About the Project

I developed this project to explore the practical use of machine learning in healthcare. It focuses on building a prediction system that can analyze patient-related information and provide a risk assessment through a simple web interface.

This project helped me gain hands-on experience in data preprocessing, model training, model deployment, and building interactive applications using Streamlit.
